Zekabot Full Render V1

Kurulum ve Kullanım:
1. Bu ZIP'i GitHub'a yükleyin ve Render'a bağlayın.
2. Render "Start Command" olarak şunu yazın: python main.py
3. Render üzerinde deploy edin.

Bu sistem GPT desteği, web scraping, API kontrolü, form girişi ve otomatik mesajlaşma gibi görevleri her dakika çalıştırır.
